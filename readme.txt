A simple version of Estatik Professional - plugin for single agent websites and real estate portals with multiple advanced options.

== Description ==

Estatik Simple plugin help you create out-of-the-box real estate website. Main features of Estatik: customizable fields (unlimited number of new fields), front-end properties
management, photo gallery, advanced search, Google map, unlimited categories/types/statuses, labels and much more! Check out http://estatik.net/ for more information.
Visit http://themes.estatik.net/ to view demo of Estatik plugin.
 
Estatik Pro features:

* frontend properties management (submitting properties after admin approval)
* agents support (agent profile, ratings, contact information)
* meta information fields 
* csv import
* social sharing features (Twitter, Facebook, Google +)
* PDF file generation
* neighbourhood information fields
* extra layout options for single property page and list (table/2 columns/list views)
* Slideshow widget
* Request information widget


== Installation ==


1. Upload files of Estatik plugin to the `/wp-content/plugins/` 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your properties via Estatik >> Add property and add them to menus.
